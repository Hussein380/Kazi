# Trusty Work

Verified workers for your home needs. Connects employers with trusted nannies, cleaners, drivers, and more.

## Features
- **Worker Profiles**: Verified backgrounds, badges, and work history.
- **Attestations**: Immutable proof of work and recommendations.
- **Job Posting**: Easy job creation for employers.
- **CV Generation**: Auto-generate professional CVs for workers.

## Tech Stack
- React + Vite
- TypeScript
- Tailwind CSS + Shadcn UI
- React Router DOM
- React Query

## Getting Started

1. **Clone the repository**
   ```bash
   git clone <YOUR_GIT_URL>
   cd trusty-work
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
